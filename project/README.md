# Final Project - FitCore Gym Fitness

## Overview
The FitCore Gym Fitness project is designed to provide an interactive and user-friendly website for gym members. It includes a member directory that allows users to view member information and toggle between different display formats.

## Project Structure
```
final-project
├── index.html
├── pages
│   └── directory.html
├── styles
│   └── style.css
├── scripts
│   ├── directory.js
│   └── app.js
├── data
│   └── members.json
└── README.md
```

## File Descriptions

- **index.html**: The main entry point of the website, detailing the site name, purpose, scenarios, color scheme, typography, and wireframe.

- **pages/directory.html**: Contains the member directory page, displaying a list of gym members and allowing users to toggle between grid and list views.

- **styles/style.css**: Contains styling rules for the website, including color schemes, typography, layout, and responsive design elements.

- **scripts/directory.js**: Responsible for fetching member data from a JSON file, displaying members in the directory, and handling view toggle functionality.

- **scripts/app.js**: Contains additional functionality for the website, such as handling user interactions and form submissions.

- **data/members.json**: Contains an array of member objects with properties like name, image, tagline, phone, address, membership level, and website.

## Setup Instructions
1. Clone the repository or download the project files.
2. Open the `index.html` file in a web browser to view the website.
3. Ensure that the `data/members.json` file is present to load member data correctly.
